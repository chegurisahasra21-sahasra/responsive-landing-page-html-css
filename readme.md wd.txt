# Responsive Landing Page

## Project Description

This project is a simple and responsive landing page created using HTML5 and CSS3. It features a modern header with navigation, a hero section with a call-to-action button, and a footer with social media links. The layout is built using Flexbox and is fully responsive with media queries for mobile devices.

## Features

* Responsive design
* Modern user interface
* Header with navigation menu
* Hero section
* Call-to-action button
* Footer with social links
* Mobile-friendly layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## How to Run

1. Download the project.
2. Open the folder in Visual Studio Code.
3. Install the Live Server extension.
4. Right-click `index.html` and select **Open with Live Server**.

## Outcome

This project demonstrates HTML structure, CSS styling, Flexbox, responsive web design, and media queries.
